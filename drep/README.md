# BEACN DRep Metadata Anchor

This folder contains the **public metadata** for the BEACN DRep registration.

## Anchor URL to use in registration

Use this raw URL as the **Anchor URL**:

- https://raw.githubusercontent.com/BEACNpool/Governance/main/drep/metadata.json

## What this is

A receipts-first DRep profile describing BEACN’s governance ethos:
- **Givers, Not Takers**
- **Public money demands public receipts**
- **Flags ≠ fraud** (signals for missing/weak proof)

## Edit policy

If this metadata is updated, it should be:
- committed to `main`
- linked from reports (optional)
- referenced in a dated snapshot when relevant
