# DRep assets (BEACNpool)

This folder contains BEACNpool’s public DRep materials and receipts.

- `BEACNpool.jsonld` — the exact JSON-LD exported from GovTool for DRep registration (byte-for-byte)
- `metadata.json` — additional metadata anchor (if used)
- `votes/` — public vote receipts and rationale reader
- `SPO-governance-delegation.md` — notes on delegating the pool reward account’s governance power to a DRep

Public vote reader:
- https://beacnpool.github.io/Governance/drep/votes/reader.html
