# BEACNpool DRep votes — 2026-02-10

This folder contains human-readable vote rationales published as immutable “receipts” for BEACNpool’s on-chain votes.

Each file includes:
- Governance Action ID
- Vote (YES/NO/ABSTAIN)
- Vote transaction hash
- Anchor URL (where applicable)
- Full rationale text

These documents are intended to mirror (or expand upon) the rationale text anchored from the vote transaction.
